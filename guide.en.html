A simple guide to programming with La

This is a very simple way to learn programming: write a little, understand quickly, and get familiar with coding logic through small examples.

1. Install

Download the official or community release from the homepage, unpack it, and try running a sample program.

2. Write your first program

// hello.la
in "Xin chào, thế giới!"

This is the simplest program: it prints a single line of text.

3. Run the program

In a terminal, run:

la1 hello.la

4. Things to remember

  • Keep each line of code short and easy to read
  • Learn input and output first
  • Try small edits and run again to learn faster

5. Common commands

To run a program, you write commands that the interpreter understands. Here are the most common ones.

5a. in

Print a value to the screen using in .... Example:

in "Xin chào, thế giới!"

5b. Variables

Use variables to store data values with đặt [name] = [value].

đặt biến = 123
// biến now holds 123

5b1. Strings

To store text, place it in quotes.

đặt chuỗi = "Xin chào, thế giới!"
// chuỗi now holds "Xin chào, thế giới!"

5b2. Numbers

To store a number, assign it directly.

đặt số = 123
// số now holds 123

5b3. Lists

To store a list, place values inside square brackets.

đặt danh_sách = [1, 2, 3, 4, 5]
// danh_sách now holds [1, 2, 3, 4, 5]

5c. Events

Event commands usually define functions, loops, or conditions and often end with kết.

5c1. tạo

Create a function with tạo [function name] ... kết.

tạo xin_chào
    in "Xin chào"
kết

xin_chào

The text "Xin chào" will be printed from the function.

5c2. lặp

Repeat a block of code with lặp [count] ... kết.

lặp 5
    in "Xin chào"
kết

You can also use lặp mãi for an endless loop, or lặp ... đến ... for conditional repetition.

5c3. nếu

Check a condition with nếu [condition] ... kết.

nếu 1 = 1
    in "Điều kiện đúng"
kết

You can also use nếu không to handle the opposite case.

5d. nhập

Read data from the keyboard with nhập.

nhập "Bạn tên là gì?"

The program pauses and waits for the user to type something.

5e. Lists

Lists hold multiple values. Use thêm, xóa, and liệt kê to manage them.

đặt danh_sách = [1, 2, 3]
thêm 4 tại danh_sách
// danh_sách now holds [1, 2, 3, 4]
xóa 2 tại danh_sách
// danh_sách now holds [1, 3, 4]
thêm 5 tại danh_sách ở 2
// danh_sách now holds [1, 3, 5, 4]
liệt kê danh_sách
// list all elements in danh_sách
