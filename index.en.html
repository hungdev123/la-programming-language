La — a simple, beginner-friendly programming language

La is designed to help beginners learn programming quickly with clear syntax and easy-to-follow documentation.

Get Started

Friendly • Easy to learn • Focused on core programming

Why choose La?

Its simple syntax, clear examples, and compact standard tools let you write real programs in just a few minutes.

Designed for learners

From annotated examples to easy-to-read errors, La lowers the language barrier for new programmers.

Main features

  • Readable and intuitive syntax
  • Friendly, easy-to-understand error messages
  • A small standard library for I/O and string handling

Who is it for?

Young learners and beginners who want a gentle introduction to programming and computer concepts.

Getting started

A small example — print a greeting with La:

// file: hello.la
in "Hello, la!"

Download the tools and run: la1 hello.la
