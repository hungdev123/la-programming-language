Hướng dẫn lập trình với ngôn ngữ "lá"

Đây là một cách học lập trình rất đơn giản: viết ít, hiểu nhanh, và làm quen với tư duy code từ những ví dụ nhỏ.

1. Cài đặt

Tải bản chính thức hoặc bản cộng đồng từ trang chủ, sau đó giải nén và chạy chương trình thử.

2. Viết chương trình đầu tiên

// hello.la
in "Xin chào, thế giới!"

Đây là một chương trình nhỏ nhất, chỉ in ra một dòng chữ.

3. Chạy chương trình

Trong terminal, chạy:

la1 hello.la

4. Những điều cần nhớ

  • Mỗi dòng code nên ngắn và dễ đọc
  • Học cách nhập và in dữ liệu trước
  • Thử sửa nhỏ rồi chạy lại để hiểu nhanh

5. Tất cả các dòng lệnh

Để chạy một chương trình, ta cần phải viết lệnh cho chương trình. Sau đây là các lệnh phổ biến nhất.

5a. in

In ra màn hình một giá trị bằng cách dùng lệnh in .... Ví dụ:

in "Xin chào, thế giới!"

5b. Biến

Sử dụng biến để lưu trữ dữ liệu bằng lệnh đặt [tên biến] = [giá trị].

đặt biến = 123
// biến bây giờ có giá trị là 123

5b1. Chuỗi

Để lưu trữ một chuỗi, ta cần đặt chuỗi trong dấu ngoặc kép.

đặt chuỗi = "Xin chào, thế giới!"
// chuỗi bây giờ có giá trị là "Xin chào, thế giới!"

5b2. Số

Để lưu trữ một số, ta chỉ cần đặt số đó vào biến.

đặt số = 123
// số bây giờ có giá trị là 123

5b3. Danh sách

Để lưu trữ một danh sách, ta cần đặt các giá trị trong dấu ngoặc vuông.

đặt danh_sách = [1, 2, 3, 4, 5]
// danh_sách bây giờ có giá trị là [1, 2, 3, 4, 5]

5c. Sự kiện

Các lệnh sự kiện thường là tạo hàm, lặp, điều kiện và thường có kết ở cuối lệnh.

5c1. tạo

Lệnh tạo dùng để tạo hàm với cú pháp tạo [tên hàm] ... kết.

tạo xin_chào
    in "Xin chào"
kết

xin_chào

Chữ "Xin chào" sẽ được in ra từ hàm.

5c2. lặp

Lệnh lặp dùng để lặp một đoạn code bằng cách lặp [số lần] ... kết.

lặp 5
    in "Xin chào"
kết

Bạn có thể dùng lặp mãi để lặp liên tục, hoặc lặp ... đến ... để dừng khi đạt điều kiện.

5c3. nếu

Lệnh nếu dùng để kiểm tra điều kiện bằng cách nếu [điều kiện] ... kết.

nếu 1 = 1
    in "Điều kiện đúng"
kết

Bạn có thể dùng nếu không để xử lý trường hợp điều kiện sai.

5d. nhập

Lệnh nhập dùng để nhận dữ liệu từ bàn phím bằng nhập.

nhập "Bạn tên là gì?"

Chương trình sẽ dừng lại và chờ người dùng nhập dữ liệu.

5e. Danh sách

Danh sách là tập hợp các giá trị dữ liệu. Bạn có thể dùng thêm, xóa và liệt kê.

đặt danh_sách = [1, 2, 3]
thêm 4 tại danh_sách
// danh_sách bây giờ có giá trị là [1, 2, 3, 4]
xóa 2 tại danh_sách
// danh_sách bây giờ có giá trị là [1, 3, 4]
thêm 5 tại danh_sách ở 2
// danh_sách bây giờ có giá trị là [1, 3, 5, 4]
liệt kê danh_sách
// liệt kê các phần tử trong danh_sách
